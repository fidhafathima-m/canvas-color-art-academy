# Canvas & Color Art Academy

A web application for art academy management with admin and student dashboards.

## Live Demo

**https://canvas-color-art-academy.vercel.app/**

## Demo Video

[How to use?](https://drive.google.com/file/d/1s9fYPcB6e_zQAp-F2gGp2JgjBvY3EkPA/view?t=2)

## Technologies Used

- **Frontend**: HTML, CSS, JavaScript, Bootstrap 5
- **Backend**: Node.js, Express.js, MongoDB
- **Authentication**: JWT
- **Deployment**: Vercel (Frontend), Render (Backend)

## Pages

- Homepage (`index.html`)
- Login Page (`pages/login.html`) 
- Admin Dashboard (`pages/admin_dashboard.html`)
- Student Dashboard (`pages/user_dashboard.html`)

## Local Development

1. Clone the repository
2. Open `index.html` with Live Server
3. Adjust style and script routes to use locally
4. Initiate node and install necessary packages
5. Run the backend
6. Access `http://localhost:3000`
